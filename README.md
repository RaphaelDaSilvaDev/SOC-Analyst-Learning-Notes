Repositório criado para documentar minha jornada de estudos em Cyber Security, com foco em Blue Team, Security Operations Center (SOC), Threat Detection e Incident Response.

O objetivo deste projeto é consolidar conhecimentos adquiridos através de laboratórios, cursos, documentação oficial, livros e plataformas práticas como o LetsDefend.

> ⚠️ Este repositório não tem como objetivo substituir materiais oficiais ou reproduzir conteúdos protegidos por direitos autorais. As informações aqui registradas representam resumos, anotações, interpretações e exemplos produzidos durante meus estudos.

---

## Objetivos

- Desenvolver conhecimento em operações de segurança (SOC)
- Melhorar o inglês técnico voltado para Cyber Security
- Criar uma base de conhecimento pessoal para consultas futuras
- Documentar laboratórios e investigações realizadas
- Construir um portfólio público de aprendizado contínuo
